# Cloud Storage Application

## Description
A cloud storage web application built using FastAPI and React. 
Users can register, login, upload files, view them, and delete them.

## Technologies Used
- Backend: FastAPI, SQLite
- Frontend: React (Vite)
- Database: SQLite
- Styling: CSS

## Features
- User Authentication
- File Upload (PDF, Images)
- View Uploaded Files
- Delete Files
- Responsive Dashboard

## How to Run
### Backend
```bash
cd backend
uvicorn app.main:app --reload
